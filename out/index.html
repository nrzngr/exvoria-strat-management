Getting Started

This is a demonstration of the Delta Force Strategy Manager. To use the full functionality:

  1. 1. Set up a Supabase project
  2. 2. Run the database migrations in database/migrations/
  3. 3. Configure the storage buckets using database/storage-setup.sql
  4. 4. Update your .env.local with your Supabase credentials
  5. 5. Start managing your Delta Force strategies!
